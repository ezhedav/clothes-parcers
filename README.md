# Clothes Parcer

Набор парсеров магазинов
Задачи падают в очередь и демон запускает скрипты  `main.py` парсеров
После сбора отдает обратно в очередь на другой демон

## Запуск скрипта

Dockerfile - еще не готов

### Rrequirements

- python 3.7
- pip3

```bash
pip3 install -r requirements.txt
```
